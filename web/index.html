Sofle Encoder Configurator

Change what the rotary knob does — clockwise, counter-clockwise, and tap.

Connect your Sofle keyboard via USB
Requirements
  • Flash the webconfig firmware (see README) before using this page.
  • Use Chrome or Edge — WebHID is required.
  • Settings are stored in the keyboard's EEPROM and survive unplugging / rebooting.
  • Default actions: Volume Down (CW), Volume Up (CCW), Mute (tap).
